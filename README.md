## Features

- Generate passwords with various character types: lowercase letters, uppercase letters, digits, and special symbols.
- Control the length of the generated password.
- Option to exclude similar-looking characters for better readability.
- Secure and random password generation using Python
- flexible to choose the type of password to generate.

## Customization

You can customize the password generation by modifying the following parameters in the `password_generator.py` script:

- `ALLOWED_CHARACTERS`: This string defines the character pool from which the password will be generated. You can add or remove characters based on your requirements.

- `EXCLUDE_SIMILAR_CHARACTERS`: Set this to `True` if you want to exclude similar-looking characters like 'l', '1', 'I', 'O', '0', etc.

- `PASSWORD_LENGTH`: Set the desired length for the generated password.
